## Installation
> Requires [Meteor](https://www.meteor.com/install), [Node](http://nodejs.org/), [npm](https://www.npmjs.com/) and [ImageMagick](http://www.imagemagick.org/script/binary-releases.php)

Developers using **Windows** should review the [Windows specific installation requirements for Meteor and Reaction](https://docs.reactioncommerce.com/reaction-docs/development/requirements).

```bash
curl https://install.meteor.com | /bin/sh # installs Meteor
git clone https://github.com/nxcong/fanshop.git
cd fanshop
meteor
```
